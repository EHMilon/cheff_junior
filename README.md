# PROJECT  OVERVIEW

This app offers a highly interactive and educational cooking experience for users, combining recipe discovery, step-by-step cooking guidance, and engaging games. Users can search for recipes, follow detailed cooking instructions with visual and voice assistance, and learn about ingredients’ origins, history, and nutritional value. The app also includes a fun, interactive mode where an avatar helps users cook, offering feedback and encouragement. Users can save favorite recipes, play cooking-related games, and track their cooking progress. The admin panel allows for recipe, user, and game management, as well as analytics and content moderation to ensure a smooth user experience.

# PROJECT PURPOSE

The purpose of this app is to make cooking fun, interactive, and educational by combining technology with culinary arts. It aims to engage users of all ages in discovering new recipes, learning about ingredients, and developing their cooking skills. The app features step-by-step guidance from an avatar, offers educational games, and includes ingredient fun facts, making cooking more enjoyable. Multilingual support ensures accessibility for a global audience. Whether you're a beginner or an experienced cook, the app provides an engaging and educational platform for culinary exploration and improvement.

# Feature list:
## User Role 
### Recipe Search & Discovery:
Search bar to find recipes.
Access to detailed recipe instructions and ingredients.
Ability to save favorite recipes.


### Interactive Recipe Steps:
Step-by-step cooking instructions with visual guides.
Voice-assisted instructions (optional).
An interactive "avatar character" guides you through the recipe.


### Ingredient Information:
Click on ingredients to view their history, origin, and nutritional value.
Fun facts about each ingredient.
Visual display of ingredients for easy identification.


### Recipe Games:
Crosswords and word searches related to recipes.
Engage with fun, learning games based on the ingredients.

### Interactive Cooking Mode:
Avatar cooks alongside the user, offering encouragement and feedback.
Option to watch the avatar cook, or cook along with the user.
Check to see if the dish is being prepared correctly.


### Ingredient Fun Facts:
Rhyming, interactive song for the recipe.
Detailed descriptions of what each ingredient does in the dish and its role in nutrition.


### Multilingual Support:
App content is available in multiple languages for accessibility.


### Materials List:
List of required materials for each recipe.
Option to check what’s missing from the pantry or shopping list.
